# Classroom Scheduler

Classroom scheduler for DEMAT (Universidad de Guanajuato).

## Installing dependences

The file `requirements.txt` is a endline-separated list of packages required for this project
```
pip install -r requirements.txt
```

## Execute project

```
python -m scheduler
```
